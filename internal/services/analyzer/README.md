# DEPRECATED - DO NOT USE

## Status: DEPRECATED

Este pacote está **depreciado** e não deve ser usado para novos desenvolvimentos.

## Alternativa

Use `internal/dsclient/stores/` para todas as operações de persistência de dados.

## Justificativa

O `dsclient` é o **Single Integration Point (SIP)** para acesso ao Data Store. Todo código novo deve usar os Stores fornecidos pelo `dsclient`.

## Futuro

Este pacote é mantido apenas para uma fase futura do projeto que envolverá análises e métricas.
